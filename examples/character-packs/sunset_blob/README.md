# Sunset Blob

Known-good sample text character pack for the Pico Buddy port.

Use this folder as the first folder-push test from Claude Desktop Hardware Buddy:

1. Open `Developer -> Open Hardware Buddy...`
2. Make sure `Claude Pico` is connected.
3. Drag this folder into the `Drop a data folder here` area.
4. Click `Send to Device`.

Expected result on the Pico:

- install progress overlay appears
- banner changes to `Installing character pack`
- the pack becomes selectable in settings and via `X`

If you want to remove it again, open:

- `Hold A` -> `settings` -> `reset` -> `delete char`
